# LNN (Latency Neural Network) — 구축 설계 명세서 v0.7

> 본 문서는 **Claude Code가 단일 명령(`/goal`)으로 전체 코드베이스를 작성**하기 위한 컨텍스트 명세입니다.
> 사람이 읽는 소개 문서가 아니라, 에이전트가 구현 대상으로 삼는 실행 사양입니다.
> 모든 설계 결정은 이미 확정되어 있으므로, 구현 중 사용자에게 되묻지 말고 본 문서의 규정을 그대로 따르십시오.
> 규정이 비어 있는 세부는 “honest fallback”(§12) 우선순위에 따라 에이전트가 판단합니다.

## 0. 목표 (Block I — Definition of Done)

> 본 명세의 `/goal` 1차 빌드 대상은 **Block I**입니다(아래 로드맵 참조). Block II·III는 인터페이스만 예약하고 구현하지 않습니다.

다음 4개 시연이 **실제 데이터로 학습되고 산출물을 남기면** Block I 성공입니다.

|# |시연        |양식                  |데이터                 |산출물              |
|--|----------|--------------------|--------------------|-----------------|
|D1|텍스트 분류    |text → class        |합성 한국어 소코퍼스         |정확도, 혼동행렬 PNG    |
|D2|이미지 분류    |image(8×8) → class  |sklearn digits (8×8)|정확도, 지형 히트맵 PNG  |
|D3|텍스트→텍스트 생성|“나는 ___를 쓴다”의 빈칸 채우기|D1 코퍼스              |빈칸 top-k 예측 출력   |
|D4|텍스트→이미지 생성|글자 토큰 “A” → 8×8 비트맵 |PIL 렌더 A–Z 8×8      |입력 글자별 8×8 출력 PNG|

세 계층 구조 **입력 Area → 처리 Cluster → 출력 Area**가 네 시연 모두에서 공통 엔진으로 동작해야 합니다.
이것은 SOTA 벤치마크가 아니라 **존재 증명(proof of concept)**입니다. 품질보다 “구조가 끝까지 흐른다”가 합격선입니다.

## 0.5 구축 로드맵 — Block I / II / III

전체 개발은 **진입 게이트를 가진 세 Block**으로 분리합니다. 앞 Block의 게이트를 통과하기 전에는 다음으로 넘어가지 않습니다. 세 Block은 **같은 코어를 공유하며 교체 가능한 모듈만 다릅니다**(§ frozen interface).

### Block I — Verification (경로 B · 존재 증명) ← **현재 빌드 대상 · 모델명 ARIS**

- **ARIS = Area-Routed Interference System.** 단일 경로 일렬(입력 Area → 처리 Area ×L → 출력 Area), Thalamus·특수 Area·분기 없음.
- 질문: “이 물리가 학습되는가?”
- 범위: 코어 동역학 + BPTT 완주, 경로 B 인코딩(작은 `D`), D1~D4, reservoir 안전망, 인코딩/디코딩 plug-in 추상화(구현은 Time만).
- **Exit gate(Block II 진입 조건)**: ① D1·D2가 reservoir를 넘어 지형 학습으로 정확도 상승, ② `test_gradient_fd` 통과, ③ `test_telescoping` 시연(선형 붕괴 vs Tobler 복원) — **정량 합격 기준은 §6 「텔레스코핑 검증」 참조**(단순 그림 산출이 아니라 분산 비교 수치가 판정). 이 셋이 녹색이 되기 전 Block II 금지.
- **안정성**: Block I은 루프 없는 DAG + 역류 금지라 동역학 행렬이 **nilpotent → 순환 발산 원천 불가**. 따라서 `ρ(A)<1` 스펙트럼 반경 정규화는 **부적용**. 감시 대상은 순환 발산이 아니라 **경로 이득 누적**(`G>1` 영역의 지수적 진폭 증가)이며 그조차 경계 φ가 Area마다 압축. 확인할 것은 “Area 내 경로 이득 폭주를 φ가 흡수하는지” 정도. (스펙트럼 반경 정규화는 나사 전위 루프가 등장하는 Block II 1급 항목.)

### Block II — Actually-Utilisable-Large-Model (경로 A · 능력 경계 측정)

- 질문: “이 구조가 규모를 감당하는가?”
- 범위: 경로 A 인코딩(FreqEncoder, 차원을 시간에서 분리), 선형 영역 운용 확립, **다중 Area 차원 분할**(단일 매질의 직교 주파수 수 한계 → 임베딩 블록 분할 운반), **Gain Field 동적 라우팅**(어텐션 유사), 잔차 스트림 도파로, 깊은 Cluster 학습 안정화.
- **대뇌 모사 구조**: Scaffolded(사람이 기능 할당·고정 배선) → Emergent 커리큘럼. Thalamus(라우팅 전담 Area), IP/RP 위상(IP를 RP 대조군으로), 특수 Area(Wernicke·Broca·Hippocampus). 상세는 `LNN_OVERVIEW.md` §3.7.
- **ε-전개 동적 라우팅**: `G=G₀+ε·δG`. δG가 **self-gating**(펄스 자신 진폭의 함수)이라야 ε-전개가 비자명(self-attention판); 외부 게이팅(다른 Area u·조건)이면 forward pass 내 상수라 완전 선형 보존(cross-attention판, ε 불요). ε는 해석↔능력↔안정성 3자 다이얼.
- **안정성(1급 — 루프 등장)**: 나사 전위 루프 → 순환계 → `ρ(A)<1` 정규화 명시 제약. self-gating은 비선형이라 단일 ρ 불가 → **일양(uniform) ρ<1**(도달 가능 상태 전체, 국소 야코비안 스펙트럼 반경) 필요.
- 성격: 엔지니어링 완주가 아니라 **연구 미해결의 집합**. 정직한 목표는 “Transformer 대체”가 아니라 **“어느 능력까지 해석 가능한 물리로 재현 가능한지의 경계 측정”**. “Large”는 파라미터 수가 아니라 “유의미한 과제를 푸는 최소 규모”로 정의.
- Exit gate: 경로 A의 차원 확장성 실증 + 능력 경계가 측정됨.

### Block III — Commercialisable-Rational-Cost-Model (능력을 비용에 맞춤)

- 질문: “능력을 합리적 연산량으로 끌어내리는가?”
- 범위(학습 중 최적화): 지형 미분 캐싱·국소지지 RBF·분리 가능 커널(§3.2 주석 참조), 저정밀·채널 가지치기·희소 지형, **하드웨어 정합**(파동·지연계는 GPU보다 아날로그/포토닉/뉴로모픽에 자연 사상 — 진짜 승부처일 수 있음).
- **추론 전용 최적화(학습 후 가능)**: 학습 후 Area 내부는 고정 LTI(선형 시불변)계이므로 격자 동역학을 매 스텝 시뮬레이션하지 않고 **(입력 Generator → 출력 Cell) 임펄스 응답 `h(t)=Σ wₚ δ(t−τₚ)`로 사전 합성**(경로 사전합성) → 추론을 합성곱 한 번으로. 비용이 `(타임스텝×변)`에서 `(입력 길이×유효 탭)`로 급감하며 **경로 분해(해석 가능성)는 오히려 명시화**. 단 경계 비선형 φ에서 끊기므로 Area 단위 적용. + 탭 가지치기(작은 진폭 곁탭 제거).
  - **분수지연 단서(정직성)**: 선형 보간 분수 지연 탭은 순수 `δ`가 아니라 **2-탭으로 번지므로**, `h(t)`는 엄밀한 임펄스 열이 아니라 각 탭이 그만큼 번진 근사다(§11의 진폭-지연 허위 결합과 같은 뿌리). Thiran 올패스 적용 시 순수 임펄스로 정화. 초판은 Thiran 미적용이므로 사전합성 `h(t)`는 근사임을 README에 명시. 상세는 `LNN_OVERVIEW.md` 추론 최적화 논의.

### Block III+ — 메모리 계층 (Block III 이후)

2축 구조: 축①감쇠율(컨텍스트 메모리 `ρ<1` 느린감쇠 ↔ Hippocampus `ρ<1` 큰감쇠, 진폭 저장) · 축②저장형식(진폭 vs 권선수). **Topological Area**(위상 보호 권선수 `∮=2πb`, `ρ` 무관, 읽기 전용 불변량 보존). 상세는 `LNN_OVERVIEW.md` Block III+.

### Block IV — Agentic (멀티모달·추론·Agent 삼위일체) ← 자리 예약(미명세)

> 추론(I~III)은 수동적 함수, Agentic은 시간에 걸친 폐루프 행동. 별도 Block. **명세는 Block II 실측 후 작성**(현재는 자리만 예약).
- **Agentic 삼각형**: 소뇌 Area(forward model = 현재+행동→미래 예측, LNN 타이밍 본질과 정합·가장 LNN다운 꼭짓점) + 기저핵(정책·억제적 선택, §3.7) + 전전두(목표·통합, §3.7).
- **폐루프 ρ(1급)**: 환경 통한 외부 루프 → ρ가 다시 1급(I nilpotent→II 나사전위 루프→IV 환경 루프의 연속). 소뇌 예측이 폐루프 안정화 장치(제어이론 안정성 ↔ LNN ρ).
- **멀티모달 결합**: Sensing Area→Integration Area(`LNN_MULTIMODAL.md`)가 Agent의 지각 입력.
- **검증 사다리**: GridWorld(격자 폐루프) → CartPole(폐루프 ρ) → Atari 저해상도(멀티모달+행동) → Minecraft/MineDojo(삼위일체 최종). 앞 단계가 무너짐의 원인 축을 격리.

### Frozen interface (세 Block 불변)

Block 경계를 넘을 때 **구현은 바뀌어도 다음 인터페이스는 동결**합니다. 이것이 각 Block이 앞 Block을 재작성하지 않고 위에 쌓게 하는 장치입니다.

- Cell 갱신 규칙(수신·합산·역류금지 분배)
- 변지연 `τ = clamp_τ(f(∇T·ê))`, `f`는 Tobler형  (※ `clamp_τ`는 구 `ρ(u)` — §3 기호 규약 참조)
- Area 경계 재생 `φ = 부호 보존 tanh`
- 경계 벡터 `u`(저차원·유부호) 통신 규약
- `encode()` / `decode()` plug-in 시그니처

> Block II로의 이행은 사실상 인코더 슬롯을 `Time→Freq`로 교체하고 Cluster 위상을 확장하는 일, Block III는 동역학·필드 평가의 *구현*만 최적화하고 *인터페이스*는 불변으로 두는 일입니다.

## 1. 아키텍처 철학 (불변 원칙)

1. 계산의 기본 단위는 행렬곱이 아니라 **격자 위 펄스의 전파·지연·간섭**입니다.
2. 파라미터는 **매질의 지형(scalar field)**이며 히트맵 한 장으로 시각화됩니다.
3. 출력은 입력 펄스가 거쳐 온 **경로 기여의 합**으로 분해 가능해야 합니다(내부 동역학은 선형 유지).
4. 비선형성은 **Area 경계의 재생(regeneration)에만** 둡니다. 이로써 Area 내부는 정확한 선형계가 되어 경로 분해·그래디언트가 깨끗하게 유지되고, “Area = 라우팅 + 경계 재생 ≈ Transformer 블록(attention + FFN/Norm)” 대응이 성립합니다.

## 2. 용어 (Glossary)

- **Cell**: 면적을 가진 육각 셀. 지연은 셀이 아니라 **여섯 변(face)**의 속성입니다(유한체적 프레임). 셀 자체는 한 스텝에 “수신·합산·포화·분배”를 수행하는 순간 노드입니다.
- **Area**: 자체 육각 격자 + 자체 지형/이득장 + 입출력 Cell 집합 + 경계 재생을 갖춘 모듈. 하나의 역할(인코딩/처리/디코딩)을 전담합니다.
- **Cluster**: Area들을 위상으로 엮은 전체 그래프. Area 사이는 **경계 벡터 `u`**(저차원, 부호 있는 스칼라 벡터)로 통신합니다.
- **Terrain `T`**: 지연을 만드는 스칼라 지형. RBF 언덕들의 가중합 + (선택) 나사 전위.
- **Gain Field `G`**: 진폭을 조각하는 양수 스칼라장(RBF 파라미터화). 진폭 라우팅·생성·어텐션 유사 기능의 전제이므로 **코어 구성요소**입니다.
- **Generator / Output Cell**: 데이터가 주입되는 입력 단자 / 진폭을 읽어내는 출력 단자.

## 3. 수학 명세 (구현 기준)

> **기호 규약(이름 충돌 방지 — 필수).** 본 명세에서 `ρ`는 **오직 스펙트럼 반경 `ρ(A)`**(§0.5 Block II+ 안정성)에만 사용합니다. 과거 판이 같은 글자를 세 곳에 쓴 충돌을 다음과 같이 분리합니다 — ① **변 지연 하한 함수**는 `clamp_τ(·)`(구 `ρ(u)`), ② **이득장 RBF 폭**은 `σ_g`(구 `ρ_k`; 지형 RBF 폭 `σ_k`와도 구별). 에이전트는 이 세 양을 코드·주석에서 같은 심볼로 쓰지 마십시오.

### 3.1 격자

- 축좌표(axial) 육각 격자, 반지름 `R`. 픽셀좌표 `x = √3·(q + r/2)`, `y = 1.5·r`.
- 방향 6개: `DIRS = [(1,0),(1,-1),(0,-1),(-1,0),(-1,1),(0,1)]`. 역방향 인덱스 `(d+3) % 6`.
- 이웃 테이블 `nbr[i*6+d]`, 경계는 `-1`(흡수).

### 3.2 지형과 기울기 (해석적)

```
T(p) = Σ_k h_k · exp(−‖p − c_k‖² / (2 σ_k²))
∇T(p) = Σ_k h_k · (−(p − c_k)/σ_k²) · exp(...)            # RBF 부분(단가)
# 나사 전위(선택, 순환/메모리용): 지형은 다가지만 기울기는 단가
∇T += Σ_m b_m · (−(y − cy_m), (x − cx_m)) / max(r_m², ε)   # ∮∇T·dℓ = 2π b_m (양자화 순환)
```

- 학습 파라미터: `h_k`(필수), 선택적으로 `c_k, σ_k, b_m`. **값 성분은 사용하지 않습니다**(순수 기울기 지연).
- **계산량 규약(Block I 필수, Block III에서 심화)**: ① 변 지연 `τ` 테이블은 지형이 롤아웃 내내 고정이므로 **`lax.scan` 바깥에서 1회 계산**한다 — 단, 그래디언트가 `τ→지형`으로 흐르도록 **미분 가능한 채로**(detach 금지) `scan` 앞에 둔다. ② RBF는 `3σ`에서 잘라 **국소 지지**로 두고 셀–언덕 버킷으로 변당 평가를 `O(1)`로 만든다(필요 시 `3σ~4σ` 코사인 윈도로 컷오프 매끄럽게). ③ 격자 기하량(변 중점·`ê`·이웃 테이블)은 학습 불변이므로 정적 상수화하고 변 차원은 `vmap`. 분리 가능 커널·1D exp 룩업은 Block III 항목.

### 3.3 변 지연 (Tobler형 비선형 법칙 — 텔레스코핑 회피)

간선 `i→j`, 변 중점 `m = (p_i + p_j)/2`, 단위방향 `ê = (p_j − p_i)/L`:

```
s = ∇T(m) · ê                                        # 변 중점 경사(부호 있음)
f(s) = τ_base · exp( γ · (|s − s*| − |s*|) )          # 볼록·비대칭, 최솟값 s*<0 (완만한 내리막 최속)
τ(i→j) = clamp_τ( f(s) )
clamp_τ(u) = τ_min + softplus(k·(u − τ_min)) / k      # 인과율 하한, 상한 τ_max로 클립 (구 ρ(u) — §3 기호 규약)
```

- **선형 법칙 금지 이유(주석으로 남길 것)**: `f(s)=a+bs`로 두면 경로 총지연이 `Σ s ≈ ∫∇T·dℓ = T(끝)−T(시작)`로 텔레스코핑되어 끝점에만 의존, 지형이 경로 간 시차를 못 만들어 학습이 붕괴합니다. 볼록 `f`는 Jensen 항으로 경사 분산이 경로를 구분하게 하여 이를 회피합니다. (정량 검증은 §6 텔레스코핑 검증.)
- `τ`는 분수값. 분수 지연은 인접 버퍼 슬롯 **선형 보간**으로 구현(초판). (개선 여지: Thiran/Lagrange — `# TODO` 주석.)

### 3.4 이득장

```
G(p) = softplus( Σ_k a_k · exp(−‖p − d_k‖²/(2 σ_g,k²)) )   # ≥ 0   (σ_g: 이득장 RBF 폭, 구 ρ_k. 지형 폭 σ_k와 구별 — §3 기호 규약)
```

분배 시 진폭에 곱합니다(아래 3.5의 4단계). `a_k=0` 초기화 → 학습 초기 이득 1.

### 3.5 셀 갱신 (한 스텝, 버퍼는 변 소속)

1. **수신**: 셀 `i`는 각 입사 변 `j→i`의 버퍼 슬롯 0 샘플을 모읍니다.
2. **중첩**: 입사 샘플을 합산. 신호가 들어온 방향 집합을 `banned`로 기록(OR).
3. **분배 준비**: 합 `a`. 허용 방향 수 `n = 6 − |banned| − (경계 변)`.
4. **방출**: 각 허용 변 `i→k`에 `per = G(p_i) · a / n`을 그 변의 `τ`에 분수 삽입. **역류 금지**(banned·경계 제외).
   - **`n=0` 보호(필수)**: 모든 방향이 banned이거나 경계라 허용 변이 없으면(`n=0`) 분배하지 않고 진폭을 **흡수(펄스 소멸)**, `per`는 미정의로 둔다. JAX에서 `0/0`은 NaN을 내고 `jax.grad`를 타고 **전체 그래디언트를 조용히 오염**시켜 학습이 죽으므로, 반드시 `per = where(n>0, G·a/n, 0)`로 보호한다. (reservoir 안전망도 같은 동역학을 쓰므로 미보호 시 동일하게 실패 → §12 fallback이 잡지 못함. 경계 셀이 많은 D2에서 특히 빈발.)
5. **포화(경계에서만)**: Area 내부는 선형. 포화/비선형은 §3.7 경계 재생에서만. (단일 Area 디버그용 `internal_saturation` 플래그 제공, 기본 `False`.)

- 펄스 파형: `wavelet(k) = sin(2π k / P)`, `P=8`, 1사이클, 평균 0.
- 극성 인코딩: 비트 0 → `+wavelet`, 비트 1 → `−wavelet`.

### 3.6 Readout

```
o(t) = Output Cell 진폭 시계열
c(t) = Σ_{k=0}^{P-1} o(t+k)·wavelet(k)                 # 정합필터
window_start = Dijkstra t_min (변지연 그래프, source→output)   # stop-gradient (※1)
score = (1/λ)·logsumexp_{t∈win}( λ·ĉ(t)² )             # λ=8, max-subtraction 안정화 (※2)
coherence_ratio = peak² / mean_background               # < 4 이면 무판정(abstain)
```

- **※1 Dijkstra는 미분 불가 → stop-gradient(필수)**: Dijkstra는 이산 그래프 알고리즘이라 그래디언트가 통과하지 못한다. `window_start`(=t_min)는 **`jax.lax.stop_gradient`로 명시 차단**하고, 학습 중 지형이 바뀌어 최적 도착 시각이 창 밖으로 이동해도 readout이 놓치지 않도록 **윈도 폭을 충분히(±몇 P) 넉넉히** 둔다. (t_min은 “어디를 적분할지”만 정하고 미분 경로엔 두지 않음 — 윈도 폭이 그 단절을 흡수.)
- **※2 logsumexp는 max-subtraction 안정화(필수)**: `ĉ²`을 지수에 넣으므로 `λ=8`이면 `exp(8·ĉ²)`가 되어 `ĉ`가 1을 조금만 넘어도 float 오버플로(inf→NaN). 반드시 표준 안정화 `logsumexp(x)=x_max+log Σ exp(x−x_max)`를 쓴다(`jax.scipy.special.logsumexp` 권장). §3.5 `n=0` 보호와 같은 “조용한 NaN” 부류 — 빠뜨리면 큰 `ĉ`에서 학습이 조용히 죽는다.
- 분류: Output Cell별 `score`에 `softmax(γ·score)`.
- 극성 부호 복원이 필요한 경우 `sign(peak)` 사용(이진·어휘 디코딩).

### 3.7 경계 재생 (Area 사이 — 유일한 비선형)

```
u_k = sign(peak_k) · |c_k(t*_k)|        # 출력 Cell k의 부호 있는 정합 점수
φ(u) = tanh(u / u0)                      # 부호 보존 압축 (유일한 비선형성)
다음 Area 재방사 진폭 = g_k · φ(u_k)     # g_k: 채널별 대각 이득(학습 가능, 밀집행렬 금지)
```

재생은 시계 리셋(타이밍 지터·번짐 소거) + 모듈 경계 + 해석 단위(`u`)를 동시에 제공합니다.

## 4. 모듈/계층 사양

### 4.1 Area

- 구성: 격자(R), 지형 T 파라미터, 이득장 G 파라미터, Generator 좌표 집합, Output Cell 좌표 집합.
- `forward(inputs) -> u`: 입력을 펄스로 방사 → `n_steps` 동안 `lax.scan` 동역학 → Output Cell readout → `u`.
- 역할 유형: `encoder` | `processor` | `decoder` (readout 모드만 다름, 엔진 공유).

### 4.2 Cluster

- Area들의 DAG(초판은 직렬 `input → processor(×L) → output`로 충분, 그래프 일반화는 인터페이스만 열어둘 것).
- Area 경계에서 `u` 전달 시 §3.7 재생 적용.
- (선택, `# TODO`) 잔차 스트림 도파로: Cluster 관통 저지연 skip 경로 + 각 Area의 읽기/쓰기 분기. 초판 미구현, 인터페이스 주석만.

### 4.3 입력 인코딩 (`encodings.py`) — plug-in 구조

**공통 인터페이스(frozen)**: `Encoder.encode(token_or_image) -> 주입 파형(들)`. 코어 동역학·readout·학습 루프는 인코더 구현에 무관해야 합니다. 플래그 `encoding="time" | "frequency"`로 교체합니다.

**핵심 사상(B·A 공통)**: 토큰 임베딩 = **토큰의 파형**. `D`차원 벡터가 곧 그 토큰 고유의 펄스 파형이고, 임베딩 학습 = 파형 학습입니다(입력·출력이 같은 코드북 공유, §4.4와 닫힘). 벡터를 파형의 *어디에* 펼치느냐가 B/A를 가릅니다.

- **`TimeEncoder` (경로 B — Block I 구현)**: 벡터 성분을 파형의 **연속 스텝**에 대응(예: `Room=(7,6,9,13)` → 스텝마다 7,6,9,13 방사). 분산 채널의 펄스 번짐을 피하기 위해 **`D` ≤ 코히어런스 길이**로 제한한다(작은 `D`에서만 유효). 작은 `D`의 본 구축에 적합하며 표준 LLM 임베딩 학습과 가장 유사.
- **`FreqEncoder` (경로 A — Block II 예약, 인터페이스만)**: 벡터 성분을 **서로 다른 주파수의 진폭**에 실어 하나의 파형으로 합성(OFDM식). 차원이 시간축이 아니라 주파수축에 겹쳐 분산 채널을 통과해도 성분 보존, 출력에서 주기별 정합필터 뱅크로 분리. **차원을 시간(롤아웃 길이)에서 분리**하므로 큰 `D` 확장의 유일한 길이며, 단일 매질 직교 주파수 수 한계 때문에 **다중 Area 차원 분할과 반드시 결합**. 선형 영역에서만 비간섭 성립(포화·강이득 시 혼변조) → Block II의 선형 운용 확립과 한 세트. Block I에서는 시그니처와 채널 직교성 단위 테스트만 두고 `# TODO`.

**임베딩 적용 — 두 층 구조와 세 결정.** 임베딩은 인코더 Area가 *내보내는* 것이 아니라 *주입받는* 입력이다. 두 층을 구분한다: **(1) 임베딩 = 토큰→파형 사전**(표준 LLM 임베딩 행렬의 LNN판, lookup table) → **(2) 인코더 Area = 그 파형을 매질에서 처리해 `u`로** 내보냄. “비슷한 단어→비슷한 펄스”는 (1)에서 성립(학습이 의미 가까운 토큰의 파형을 — 경로 B면 시간 모양, 경로 A면 주파수 스펙트럼 — 닮게 함). LNN 고유의 의미 유사도 = **(a) 파형 유사도(코사인의 LNN판) × (b) 매질의 위상 정렬 라우팅** — 표준 임베딩엔 (a)만, LNN은 매질이 (b)를 더함. 세 결정:

1. **`D`(=파형 길이) 작게**: 경로 B는 `D ≤ 코히어런스 길이`라 LNN 임베딩은 표준(수백~수천)과 달리 **한 자릿수~십 단위**. (§11 D3 충돌의 뿌리)
2. **매질과 함께 학습**: reservoir 단계에선 임베딩도 고정, **지형 개방 단계에서 임베딩+매질을 동시 학습**(파형과 그 라우팅이 정합해야 함). `optax.multi_transform` 파라미터 그룹에 임베딩 포함.
3. **입출력 코드북 공유(weight tying의 LNN판)**: 입력 파형 사전 = 출력 vocabulary 코드 `c_v`. 출력 Area가 어휘 `v`를 가리키는 코드가 곧 입력에서 `v`를 주입하는 파형이어야 인코딩-디코딩 일관(§4.4와 닫힘).
   → 학습 후 LNN 의미론 전체가 **“파형 사전 + Area당 K개 임펄스 응답 함수”** 두 산출물로 읽힘(표준 LLM의 임베딩 행렬+가중치 행렬에 대응하되 양쪽 다 해석 가능).

- **이미지**: 픽셀 → Generator 공간 대응. 밝기 → 진폭(기본) 또는 주입 위상(에지 강조, 플래그). 동시 방사 → 초기 파면 = 이미지 구조.
- **텍스트**: 토큰을 일정 스텝 간격으로 순차 주입. 위치는 별도 파라미터 없이 **주입 시점**으로 인코딩(지연선 일치 검출이 상대 위치를 잡음).

### 4.4 출력 디코더 (`decoders.py`) — 세 모드 · plug-in 구조

**공통 인터페이스(frozen)**: `Decoder.decode(readout) -> 출력`. 인코더와 같은 코드북을 공유(특히 vocabulary 모드).

- **discriminative**: 클래스당 Output Cell + §3.6 score + softmax. (D1, D2)
- **structural**: Output Cell 격자를 출력 좌표(픽셀/위치)에 공간 대응, 각 셀 정합 응답 = 그 위치 값. (D4, 8×8 캔버스)
- **vocabulary (CDMA식)**: 어휘마다 (준)직교 코드 `c_v` 배정. 출력 Area는 `m ≈ ceil(log2|V|)`개 채널에 코드 성분 방출. 디코딩 = 코드북 정합필터 상관의 argmax = `softmax over vocabulary`의 물리 등가. (D3) — 경로 A에서는 코드가 곧 주파수 채널이 되어 인코더와 자연 정합.

## 6. 학습 사양 (`train.py`)

- **프레임워크**: JAX + optax (+ equinox로 모듈/파라미터 PyTree). 동역학은 `jax.lax.scan`, BPTT는 `jax.grad` + `jax.checkpoint`(remat)로 스텝 메모리 절감.
- **파라미터 그룹별 학습률**: `optax.multi_transform`으로 `{terrain_h, gain_a, embedding, decoder_code, diag_gain}` 분리(사용자 기존 워크플로와 정합).
- **커리큘럼(필수 구현)**:
  1. **Reservoir 모드 기준선**: 지형/이득 고정, readout·디코더만 학습 → 반드시 동작하는 하한선 확보.
  2. **코히어런스 어닐링**: wavelet 사이클 수를 길게(매끄러운 손실) 시작해 점차 짧게.
  3. **지형 개방**: terrain `h_k` → (선택) gain → (선택) `c_k,σ_k`.
  4. **경계별 탐욕 사전학습** 후 엔드투엔드 미세조정(깊은 BPTT의 기울기 사막 완화).
- **손실**: 분류=교차엔트로피(softmax score). 생성(D4)=MSE. D3=마스크 토큰 교차엔트로피. 초기 보조손실로 정답 Output Cell에 목표 wavelet 형태를 맞추는 파형 감독(선택, 기울기 조밀화).
- **그래디언트 검증(`tests/test_gradient_fd.py`)**: fractional-delay 사슬에 대한 유한차분 대조 테스트. 합격 기준: 해석적 그래디언트와 유한차분의 상대오차 < `1e−3`(대표 파라미터 표본).
- **텔레스코핑 검증(`tests/test_telescoping.py`) — 정량 합격 기준(Exit gate ③)**: 동일 끝점·상이 경로 쌍 `N ≥ 20`개에 대해 각 경로의 총지연 `Στ`를 측정하고, **무차원 상대분산** `V_rel = Var(Στ) / (Δτ_endpoint)²`로 판정한다(`Δτ_endpoint` = 두 끝점 간 최소경로 지연차; 하이퍼파라미터 `τ_base`·`R`이 바뀌어도 임계가 불변이도록 무차원화 — 절대분산 임계 금지).
  - **선형** `f(s)=a+bs`: `V_rel < ε_rel` (텔레스코핑 확인 — 총지연이 끝점에만 의존, 경로 간 차이 소멸).
  - **Tobler** `f`: `V_rel > c · ε_rel`, `c ≥ 10` (경사 분산이 경로를 구분 — 복원 확인).
  - **두 조건 동시 충족 시 통과.** `ε_rel`은 부동소수 잡음 수준의 무차원 임계 `~1e−6`(상대분산이므로 스케일 불변).
  - 주의: 이 테스트의 판정은 “그럴듯한 시연 그림”이 아니라 **위 무차원 분산 비교 수치**다. 그림(`outputs/`)은 보조 산출물로만 저장한다.

### 6.1 계산량 최적화 메모

- §3.2 계산량 규약(τ 1회 계산·국소지지 RBF·정적 기하량·`vmap`)을 학습에서 그대로 적용. 분리 가능 커널·저정밀은 Block III.

### 6.2 모델 구성 명기 (필수 — 모든 시연 결과에 기록)

> Experiment 1은 정확도만 기록하고 "어떤 크기의 모델이 그 정확도를 냈는가"를 빠뜨려 사후 추적이 불가했다. **모든 시연의 결과 JSON(`*_metrics.json`)과 `report.md`에 아래 구성을 빠짐없이 기록**한다. `config_of(model)` 유틸로 자동 수집하고 사람이 수기 입력하지 않는다 — 정확도·비용 해석의 필수 메타데이터다.

- **Area별**: `R`(격자 반지름), `N_cells`(셀 수), `N_edges`(변 수 ≈ 3·N_cells), Generator 좌표 수, Output Cell 좌표 수, RBF 언덕 수(지형 `K_terrain`·이득 `K_gain`), 역할(encoder/processor/decoder).
- **Cluster**: Area 개수·종류, processor 수 `L`, 위상(serial/분기).
- **인코더**: 종류(Time/Freq), 임베딩 차원 `D`, 어휘 수 `|V|`, 주입 방식(공간/시간), 코히어런스 길이(사이클).
- **디코더**: 종류(discriminative/structural/vocabulary), 출력 채널 수 `m`, 코드북 공유 여부.
- **롤아웃**: `n_steps`, 펄스 주기 `P`, batch.
- **파라미터 수**: 그룹별(`terrain_h`·`gain_a`·`embedding`·`decoder_code`·`diag_gain`) 학습 파라미터 개수와 **총합** — 표준 신경망 "파라미터 수"에 대응하는 LNN의 정직한 등가물(비용·규모 비교의 기준).

## 7. 생성 메커니즘 (D3, D4)

- D3(텍스트→텍스트): vocabulary 디코더로 `<MASK>` 위치 예측. 다양성이 필요하면 주입 위상에 잡음 시드(`phase noise = latent`).
- D4(텍스트→이미지): 글자 토큰 조건 → processor → structural 디코더 8×8. (선택 고급) 시간 역전 생성·정상파 어트랙터는 `# TODO` 인터페이스만 두고 초판은 지도학습 디코더로 충분.
- **복원/생성 모드 주의(Block I은 nilpotent)**: Block I은 루프가 없어 **자유 전개형 어트랙터 복원**(입력 끊고 분포 전형으로 수렴)은 0으로 자명 수렴해 무의미 — 나사 전위 루프가 있는 Block II+ 전용. Block I의 D3·D4 생성은 반드시 **지속 구동형**(조건 입력을 매 스텝 계속 걸고 정상 응답을 readout)으로 구현한다. zero-mean 구동에서 정상 응답이 자명해지지 않도록 구동 파형을 설계할 것. (“조건→출력 장”이 곧 지속 구동형.)

## 11. 알려진 한계 (README에 명시할 것)

- 코히어런스 길이(사이클 수)가 직접 간섭 가능한 토큰 간 거리의 상한 → 장문맥 불가. 본 구축은 단문 빈칸·소규모 생성만 목표.
- 선형 보간 분수 지연의 진폭-지연 허위 결합 잔존(Thiran 미적용). → Block III 추론 사전합성 `h(t)`도 이 때문에 근사(§0.5 분수지연 단서).
- 진폭 라우팅은 Gain Field에 의존하며 완전 동적 어텐션은 미구현(약화된 라우팅).
- 전 과정은 **존재 증명**이며 현대 LLM/비전 모델과 성능 비교 대상이 아님.
- BPTT 비볼록성·기울기 사막 가능 → reservoir 기준선이 안전망.
- **경로 B의 차원-시간 결합(Block II가 경로 A로 가야 하는 이유)**: 경로 B에서 임베딩 차원 `D`는 곧 파형 길이=시간 예산이다. `D`를 키우면 ① 롤아웃 스텝↑, ② 파면을 담을 격자(변 수)↑, ③ 코히어런스 길이↑로 간섭 선택성(=해석 가능성)↓ 가 **동시에** 일어나, 비용이 `D`에 초선형으로 붙는다(표준 LLM은 `D²`지만 시간과 무관, 여기선 시간을 직접 잡아먹음). 따라서 경로 B의 `D`는 본질적으로 작게 유지해야 하며, 큰 `D`는 경로 A(차원↔시간 분리) + 다중 Area 차원 분할로만 가능하다. 경로 A는 단순 견고성 개선이 아니라 **계산량 폭증을 막는 필수 확장**.
- **★ 가장 먼저 깨질 가정(빌드 시 주시)**: D3(빈칸 채우기)의 vocabulary 디코더는 `D ≈ log₂|V| + 마진 ≈ 8`을 요구하는데, 경로 B는 `D ≤ 코히어런스 길이`를 강제하므로 **코히어런스 길이 ≥ 8 ≈ 펄스 주기 P**가 되어 간섭 선택성(해석 가능성)이 무너지는 충돌이 예상됨. 게이트①의 정확도 저하 이전에 **인코더 단위 테스트의 채널 직교성 붕괴**(코드 간 최대 교차상관이 임계 초과)로 먼저 드러날 가능성. D1·D2(작은 K)는 충돌 없이 통과하되 **D3·D4에서 선택적으로** 나타날 것 → 그 경우 경로 A(FreqEncoder)를 Block II가 아니라 **Block I 후반으로 앞당기라는 첫 경험적 신호**. 빌드 시 D3 인코더 직교성을 우선 측정할 것(§9 5단계·§10 비고와 연결).

## 12. Fallback 우선순위 (사양이 빌 때)

1. **완주 > 품질**: 모든 시연은 무엇이든 가시적 출력을 남긴다. 완전 학습이 안 되면 reservoir 모드(고정 지형 + 학습 readout) 결과라도 저장한다.
2. **코어 정확성 > 기능 범위**: 엔진과 그래디언트 검증이 기능 확장보다 우선. `# TODO`로 미룰 것: 잔차 도파로, 시간역전 생성, 정상파 어트랙터, Thiran 분수지연, 그래프 일반 Cluster.
3. **결정론**: 모든 데이터·초기화에 고정 시드.
4. **CPU 우선**: GPU 없이도 완주 가능해야 한다.
5. **작게 시작**: 막히면 R·스텝·데이터·epoch를 줄여 파이프라인을 먼저 닫고 점증한다.

> 주의: 다음 넷은 fallback 대상이 아니라 **무조건 준수**한다 — ① `n=0` 분배 보호(§3.5), ② readout의 Dijkstra stop-gradient·logsumexp max-subtraction 안정화(§3.6), ③ `clamp_τ`/`σ_g` 기호 분리(§3). ①②는 NaN으로 학습을 조용히 죽이고, ③은 심볼 충돌로 잘못 구현되어도 조용히 통과 보고될 수 있어 가장 비싼 실수다.
