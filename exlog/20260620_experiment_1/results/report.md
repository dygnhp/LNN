## Q1~Q4 판정

- **Q1 (기하 바닥)**: orth_max(MAX) D8=0.995 → D32=0.983 — 거의 불변(미해소). **그러나 within(범주 내
  평균)은 0.946 → 0.903 → 0.803 로 단조 하강** — D 가 코드 분리의 *여지*는 줌. MAX 가 안 풀리는 이유는
  Exp2 결론과 정합: 같은 범주 명사가 동사를 공유해(데이터 보상 구조) **최악의 한 쌍은 차원이 남아돌아도
  분리를 보상받지 못해** colinear 유지 → 기하 바닥은 D 의 문제가 아니라 **데이터의 문제**임이 차원 축에서
  재확인(Exp2 H-ORTH 와 교차 확증). within/across(0.803 ≫ 0.306)는 D 키워도 의미 구조 보존.
- **Q2 (라우팅 병목 O4)**: top1 D8=0.417 → D32=0.479. **상승** — 코드 분리가 top1 로 활용됨.
- **Q3 (의미 구조)**: within≥across 모든 D 유지 True (D32: within=0.803 ≥ across=0.306).
- **Q4 (천장 0.56) [핵심]**: best acc_mnist=0.435 vs 0.56. **미돌파 — ~0.56 은 차원과도 무관한 더 깊은 물리 한계(4번째 독립 확증: 용량 Exp3 / 구조 Phase1 / 완전통합 Phase2 / 차원 Phase3).**
